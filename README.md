## Getting credentials

Follow instructions [here](https://github.com/danthareja/node-google-apps-script/blob/master/README.md).

Save a file called drive-credentials.json to your instance folder.

```bash
# install node packages locally

npm install
# run this
# follow instructions in your web browser
# and enter obtained code in terminal
./node_modules/.bin/gapps auth instance/drive-credentials.json

./node_modules/.bin/gapps init <the-script-id>
```

You should be ready to go.
